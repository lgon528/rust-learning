# 示例 1：枚举和模式匹配

本示例旨在演示 Rust 的枚举（Enums）和模式匹配（Pattern Matching），它们是 Rust 中用于处理不同可能性和控制程序流程的强大工具。

## 核心概念

### 1. 枚举（Enums）

枚举是一种允许你定义一个可以包含多种不同变体（variants）的类型。每个变体都可以有关联的数据。

- **定义枚举**：使用 `enum` 关键字定义，每个变体可以有不同的数据类型和数量。
- **方法**：可以在枚举上使用 `impl` 块来定义方法，就像在结构体上一样。

### 2. 模式匹配（Pattern Matching）

模式匹配是 Rust 中一种强大的控制流结构，它允许你根据值的结构来执行不同的代码。

- **`match` 表达式**：`match` 关键字允许你将一个值与一系列模式进行比较，并根据匹配的模式执行相应的代码。`match` 表达式必须是穷尽的，即必须覆盖所有可能的情况。
- **`if let` 表达式**：`if let` 是一种更简洁的语法，用于只关心一种匹配情况而忽略其他情况的场景。它等价于一个只有一个匹配臂的 `match` 表达式。

## 如何运行

你可以使用以下命令来运行本示例：

```bash
cargo run -p enums-and-pattern-matching-demo
```

该命令将编译并执行 `src/main.rs` 文件，你将在终端看到关于枚举和模式匹配的演示输出。
