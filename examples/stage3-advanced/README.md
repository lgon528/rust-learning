# 第 3 阶段：高级概念

欢迎来到 Rust 学习计划的第 3 阶段！在本阶段，我们将深入探讨 Rust 的一些高级概念，这些概念将帮助你编写更强大、更灵活的代码。

## 学习目标

在本阶段结束时，你应该能够：

-   理解并使用枚举和模式匹配来处理不同的可能性。
-   掌握 Rust 的高级错误处理技术，包括自定义错误类型和 `?` 运算符。
-   理解并使用智能指针来管理内存和实现内部可变性。

## 示例项目

为了帮助你掌握这些概念，我们提供了一系列示例项目。每个项目都侧重于一个特定的主题，并包含详细的解释和代码示例。

-   **[示例 1：枚举和模式匹配](./01-enums-and-pattern-matching)**：演示如何使用枚举和模式匹配来处理不同的可能性。
-   **[示例 2：高级错误处理](./02-error-handling)**：介绍如何创建自定义错误类型、使用 `?` 运算符以及如何让 `main` 函数返回 `Result`。
-   **[示例 3：智能指针](./03-smart-pointers)**：演示如何使用 `Box<T>`、`Rc<T>` 和 `RefCell<T>` 等智能指针。

## 如何开始

建议你按顺序学习每个示例项目。对于每个项目，请先阅读 `README.md` 文件以了解其核心概念，然后查看 `src/main.rs` 文件中的代码示例。你可以使用 `cargo run` 命令来运行每个项目，并观察其输出。
