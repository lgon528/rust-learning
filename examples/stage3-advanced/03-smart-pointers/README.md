# 示例 3：智能指针

本示例旨在演示 Rust 的智能指针，它们是实现了 `Deref` 和 `Drop` trait 的结构体。智能指针提供了比普通引用更强大的功能。

## 核心概念

### 1. `Box<T>`

`Box<T>` 是一种在堆上分配值的智能指针。它最常用于以下场景：

- 当你有一个在编译时无法确定大小的类型，并且希望在需要时能够存储该类型的值。
- 当你有一个大的数据结构，并且希望在不复制数据的情况下转移其所有权。
- 当你希望拥有一个值，但只关心它是否实现了某个特定的 trait，而不是它的具体类型。

### 2. `Rc<T>`

`Rc<T>` 是一种引用计数的智能指针，它允许多个所有者。当你希望在程序的多个部分之间共享数据时，可以使用 `Rc<T>`。当最后一个所有者离开作用域时，`Rc<T>` 和它指向的数据都会被释放。

### 3. `RefCell<T>`

`RefCell<T>` 是一种在运行时强制执行借用规则的智能指针。它允许你在拥有不可变引用的同时修改数据，这被称为“内部可变性”。`RefCell<T>` 在编译时不会检查借用规则，而是在运行时检查。如果违反了借用规则，程序将会 panic。

## 如何运行

你可以使用以下命令来运行本示例：

```bash
cargo run -p smart-pointers-demo
```

该命令将编译并执行 `src/main.rs` 文件，你将在终端看到关于智能指针的演示输出。
