# 第 2 阶段：所有权系统

在 Rust 中，所有权系统是其最核心和最独特的特性之一。它是一组规则，用于管理内存，确保在编译时就能发现内存安全问题，而无需垃圾回收器（GC）。

本阶段将深入探讨所有权系统的三个核心概念：

1.  **所有权（Ownership）**：每个值都有一个被称为其 *所有者* 的变量。在任何时候，一个值只能有一个所有者。当所有者离开作用域时，该值将被丢弃。
2.  **借用（Borrowing）**：允许你在不获取所有权的情况下访问数据。借用分为不可变借用（`&T`）和可变借用（`&mut T`）。
3.  **生命周期（Lifetimes）**：确保所有借用都是有效的。生命周期是 Rust 编译器用来检查引用有效性的一种方式。

## 学习目标

完成本阶段后，你将能够：

- 理解所有权、移动和克隆的语义。
- 掌握不可变和可变借用的规则。
- 理解生命周期如何防止悬垂引用。
- 使用结构体和方法来组织代码。
