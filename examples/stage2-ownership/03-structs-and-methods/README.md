# 示例 3：结构体和方法

本示例旨在演示 Rust 的结构体（Structs）和方法（Methods），它们是 Rust 中创建自定义数据类型和组织相关功能的核心机制。

## 核心概念

### 1. 结构体（Structs）

结构体是一种复合数据类型，允许你将多个不同类型的值组合成一个有意义的整体。在 Rust 中，结构体有三种形式：

- **经典结构体（Classic Struct）**：包含一组命名字段。
- **元组结构体（Tuple Struct）**：字段没有名称，只有类型。
- **单元结构体（Unit-like Struct）**：没有任何字段，通常用于泛型编程或标记。

本示例主要关注经典结构体。

### 2. 方法（Methods）

方法是与结构体（或枚举、trait）关联的函数。它们在 `impl`（implementation）块中定义，并且第一个参数通常是 `self`、`&self` 或 `&mut self`，代表结构体的实例。

- `&self`：表示该方法不可变地借用结构体实例。
- `&mut self`：表示该方法可变地借用结构体实例。
- `self`：表示该方法获取结构体实例的所有权。

### 3. 关联函数（Associated Functions）

关联函数也定义在 `impl` 块中，但不以 `self` 作为第一个参数。它们与结构体相关，但不作用于特定的实例。最常见的关联函数是构造函数，通常命名为 `new` 或其他描述其功能的名称。

## 如何运行

你可以使用以下命令来运行本示例：

```bash
cargo run -p structs-and-methods-demo
```

该命令将编译并执行 `src/main.rs` 文件，你将在终端看到关于结构体和方法的演示输出。
