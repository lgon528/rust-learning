# 示例 2：使用 Tokio 进行异步编程

本示例旨在演示如何使用 `tokio` 库在 Rust 中进行异步编程。

## 核心概念

### 1. `tokio`

`tokio` 是一个用于编写异步应用程序的运行时。它提供了一些核心组件，例如：

-   一个多线程的、工作的窃取任务调度器。
-   一个由 `mio` 驱动的 I/O 事件循环。
-   一个计时器 API，用于调度在将来的某个时间点执行的代码。

### 2. `async`/`await`

`async`/`await` 是 Rust 中用于编写异步代码的语法。`async` 关键字用于将一个函数标记为异步函数，而 `await` 关键字用于等待一个 `Future` 的完成。

### 3. `tokio::spawn`

`tokio::spawn` 函数用于在 `tokio` 运行时上生成一个新的异步任务。该函数返回一个 `JoinHandle`，你可以使用它来等待任务的完成。

## 如何运行

你可以使用以下命令来运行本示例：

```bash
cargo run -p tokio-async-demo
```

该命令将编译并执行 `src/main.rs` 文件，你将在终端看到关于异步编程的演示输出。
