# 练习：高级错误处理

这个练习将帮助你深入学习 Rust 的高级错误处理技术。

1.  **自定义错误类型**：学习如何创建自己的错误类型。
2.  **错误处理库**：了解 `thiserror` 和 `anyhow` 等流行的错误处理库。
3.  **错误恢复**：学习如何从错误中恢复，而不是简单地传播它们。

## 任务

- [ ] 创建一个新项目 `advanced-error-handling-practice`。
- [ ] 定义一个自定义的错误枚举，用于表示应用程序中可能出现的不同错误。
- [ ] （可选）使用 `thiserror` 库来简化自定义错误类型的创建。
- [ ] （可选）使用 `anyhow` 库来轻松地处理和返回错误。
