# 练习：比较

这个练习将帮助你熟悉 Rust 中的比较运算符和可比较的 trait。

1.  **比较运算符**：学习如何使用 `==`、`!=`、`<`、`>`、`<=` 和 `>=`。
2.  **`PartialEq` 和 `Eq`**：了解如何为自定义类型实现这些 trait 以进行比较。
3.  **`PartialOrd` 和 `Ord`**：了解如何为自定义类型实现这些 trait 以进行排序。

## 任务

- [ ] 创建一个新项目 `comparisons-practice`。
- [ ] 比较不同的基本数据类型的值。
- [ ] 创建一个 `Point` 结构体，并为其实现 `PartialEq` 和 `PartialOrd`。
- [ ] 创建一个 `Point` 的向量，并使用 `sort` 方法对其进行排序。
