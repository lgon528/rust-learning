# 练习：所有权入门

这个练习将帮助你理解 Rust 的核心概念：所有权。

1.  **所有权基础**：理解变量和数据如何交互，以及所有权如何转移。
2.  **引用和借用**：学习如何使用引用来访问数据而无需获取其所有权。
3.  **切片**：学习如何引用集合的一部分。
4.  **生命周期**：了解生命周期如何确保引用始终有效。

## 任务

- [ ] 创建一个新项目 `ownership-practice`。
- [ ] 编写一个函数，接收一个 `String` 的所有权，并打印它。
- [ ] 编写一个函数，接收一个 `String` 的引用，并计算其长度。
- [ ] 编写一个函数，接收一个字符串切片，并返回第一个单词。
- [ ] （可选）尝试编写一个返回悬垂引用的函数，并观察编译器如何阻止你这样做。
