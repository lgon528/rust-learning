# 练习：环境配置

这个练习将帮助你熟悉 Rust 开发环境的配置，包括：

1.  **安装 Rust**：验证你的 Rust 安装是否成功。
2.  **使用 Cargo**：学习如何使用 Cargo 创建、构建和运行一个新的 Rust 项目。
3.  **Hello, World!**：编写并运行你的第一个 Rust 程序。

## 任务

- [ ] 检查 Rust 和 Cargo 的版本。
- [ ] 使用 `cargo new` 创建一个名为 `hello-rust` 的新项目。
- [ ] 在 `main.rs` 文件中，编写一个打印 “Hello, Rust!” 的程序。
- [ ] 使用 `cargo run` 编译并运行你的程序。
- [ ] 使用 `cargo check` 检查代码是否有错误。
- [ ] 使用 `cargo build` 构建项目的可执行文件。
