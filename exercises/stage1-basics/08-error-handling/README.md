# 练习：错误处理

这个练习将帮助你学习 Rust 的错误处理机制。

1.  **`panic!`**：了解何时以及如何使用 `panic!` 处理不可恢复的错误。
2.  **`Result`**：学习如何使用 `Result` 枚举处理可恢复的错误。
3.  **错误传播**：学习如何使用 `?` 运算符传播错误。

## 任务

- [ ] 创建一个新项目 `error-handling-practice`。
- [ ] 编写一个可能导致 `panic!` 的函数（例如，访问数组越界）。
- [ ] 编写一个函数，尝试打开一个文件，并使用 `Result` 处理可能出现的错误。
- [ ] 编写一个函数，读取文件内容，并使用 `?` 运算符将错误传播给调用者。
