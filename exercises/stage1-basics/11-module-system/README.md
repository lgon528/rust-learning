# 练习：模块系统

这个练习将帮助你掌握 Rust 的模块系统，用于组织和管理代码。

1.  **模块基础**：学习如何使用 `mod` 关键字创建模块，以及如何使用 `use` 和 `pub` 控制可见性。
2.  **Cargo 和包管理**：了解 Cargo 如何与模块系统协同工作。
3.  **工作空间**：学习如何管理包含多个包的大型项目。

## 任务

- [ ] 创建一个新项目 `module-system-practice`。
- [ ] 将你的代码组织到不同的模块中（例如，`front_of_house` 和 `back_of_house`）。
- [ ] 使用 `use` 关键字将模块导入作用域。
- [ ] 使用 `pub` 关键字使模块的某些部分公开。
- [ ] （可选）将你的项目转换为一个包含多个包的工作空间。
