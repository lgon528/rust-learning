# 练习：基本语法

这个练习将帮助你熟悉 Rust 的基本语法，包括：

1.  **变量和可变性**：学习如何声明变量，以及如何使用 `mut` 关键字使其可变。
2.  **数据类型**：了解 Rust 的基本数据类型，如整数、浮点数、布尔值和字符。
3.  **函数**：学习如何定义和调用函数。
4.  **注释**：学习如何编写代码注释。

## 任务

- [ ] 创建一个新项目 `syntax-practice`。
- [ ] 在 `main.rs` 中，声明一个不可变变量和一个可变变量，并打印它们的值。
- [ ] 声明不同基本数据类型的变量，并打印它们的值。
- [ ] 编写一个函数，接收两个整数作为参数，并返回它们的和。在 `main` 函数中调用这个函数并打印结果。
- [ ] 为你的代码添加注释，解释每个部分的功能。
