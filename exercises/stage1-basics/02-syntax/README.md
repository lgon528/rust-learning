# 练习 2：基本语法

## 目标

练习 Rust 的基本语法，包括变量、数据类型和函数。

## 任务

1.  打开 `exercises/stage1-basics/02-syntax/exercise/src/main.rs` 文件。
2.  修改 `main` 函数，使其包含以下内容：
    *   一个名为 `x` 的可变整数变量，并将其初始化为 `5`。
    *   将 `x` 的值重新分配为 `6`。
    *   一个名为 `tup` 的元组，包含一个整数、一个浮点数和一个字节。
    *   一个名为 `a` 的数组，包含五个整数。
3.  在 `main` 函数的末尾，添加对 `println!` 宏的调用，以打印 `x`、`tup` 的第二个元素和 `a` 的第一个元素。
4.  使用 `cargo run -p exercise` 运行项目，并验证输出是否正确。

## 验收标准

-   `cargo run -p exercise` 命令执行成功，没有任何错误。
-   控制台输出 `x`、`tup` 的第二个元素和 `a` 的第一个元素的值。
