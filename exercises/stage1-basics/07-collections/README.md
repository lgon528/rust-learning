# 练习：集合

这个练习将帮助你熟悉 Rust 的常用集合类型。

1.  **向量（Vector）**：学习如何创建、更新和读取向量的元素。
2.  **哈希映射（Hash Map）**：学习如何使用哈希映射存储键值对。
3.  **其他集合**：简要了解 `String`、`HashSet` 等其他集合类型。

## 任务

- [ ] 创建一个新项目 `collections-practice`。
- [ ] 创建一个整数向量，并向其中添加一些元素。
- [ ] 遍历向量并打印每个元素。
- [ ] 创建一个哈希映射，用于存储学生姓名和分数。
- [ ] 从哈希映射中查找并打印特定学生的分数。
- [ ] 编写一个函数，接收一个字符串，并返回其中每个单词出现的次数。
