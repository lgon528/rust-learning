# 练习：结构体和枚举

这个练习将帮助你掌握 Rust 的复合数据类型：结构体和枚举。

1.  **结构体定义**：学习如何定义和实例化结构体。
2.  **方法和关联函数**：学习如何为结构体定义方法和关联函数。
3.  **枚举定义**：学习如何定义和使用枚举。
4.  **模式匹配**：学习如何将 `match` 与枚举一起使用。

## 任务

- [ ] 创建一个新项目 `structs-enums-practice`。
- [ ] 定义一个 `Rectangle` 结构体，包含 `width` 和 `height` 字段。
- [ ] 为 `Rectangle` 实现一个计算面积的 `area` 方法。
- [ ] 定义一个 `Shape` 枚举，包含 `Circle`、`Square` 和 `Rectangle` 等变体。
- [ ] 编写一个函数，接收一个 `Shape`，并使用 `match` 计算其面积。
