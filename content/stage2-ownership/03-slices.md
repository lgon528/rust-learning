# 切片

切片（slice）允许我们引用集合中连续的元素序列，而不是整个集合。切片是一种引用，因此它没有所有权。

## 字符串切片

字符串切片是对 `String` 的一部分的引用。

```rust
let s = String::from("hello world");

let hello = &s[0..5];
let world = &s[6..11];
```

字符串字面值是切片。`let s = "Hello, world!";` 中 `s` 的类型是 `&str`：它是一个指向二进制程序特定位置的切片。这也就是为什么字符串字面值是不可变的；`&str` 是一个不可变引用。

## 其他切片

字符串切片是特定于字符串的。但也有更通用的切片类型。考虑一个数组：

```rust
let a = [1, 2, 3, 4, 5];

let slice = &a[1..3];

assert_eq!(slice, &[2, 3]);
```

这个切片的类型是 `&[i32]`。它的工作方式与字符串切片相同，通过存储指向第一个元素的指针和长度。
