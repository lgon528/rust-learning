# Web 框架

Rust 拥有多个用于构建 Web 应用程序的框架。以下是一些最受欢迎的框架：

## `actix-web`

`actix-web` 是一个功能强大、实用且速度极快的 Web 框架。它建立在 `actix` actor 框架之上，并提供了类型安全、功能丰富的 API。

## `rocket`

`rocket` 是一个用于 Rust 的 Web 框架，它使编写快速、安全的 Web 应用程序变得简单，而不会牺牲灵活性、可用性或类型安全。它的主要卖点是其富有表现力的路由和易于使用的请求处理。

## `axum`

`axum` 是一个符合人体工程学且模块化的 Web 框架，由 `tokio` 团队构建。它与 `tokio` 生态系统紧密集成，并专注于提供一组用于构建可扩展、可靠的 Web 服务的核心功能。

## `warp`

`warp` 是一个可组合的、类型安全的 Web 服务器框架。它允许你通过将较小的“过滤器”组合在一起来构建强大的 Web 服务。
