# 项目3：构建简单的区块链

本项目演示了如何构建一个非常简单的区块链。区块链是一个分布式、不可变的账本，用于记录交易。

## 核心概念

- **区块**：区块链的基本构建块，包含数据、时间戳和前一个区块的哈希。
- **区块链**：一系列链接在一起的区块。
- **哈希**：用于确保区块链完整性的加密哈希函数。

## 运行项目

要运行此项目，请导航到 `examples/stage5-projects/03-blockchain-demo` 目录并运行以下命令：

```bash
cargo run
```

该程序将创建一个新的区块链，向其中添加两个区块，然后将它们打印到控制台。
