# 项目1：使用 Axum 构建 Web 应用

本项目演示了如何使用 `axum` 框架构建一个简单的 Web 应用程序。`axum` 是一个符合人体工程学且模块化的 Web 框架，由 `tokio` 团队构建。

## 核心概念

- **路由**：使用 `axum::Router` 来定义应用程序的路由。
- **处理程序**：处理程序是响应请求的异步函数。
- **提取器**：提取器用于从请求中提取数据，例如路径参数、查询字符串或请求正文。
- **响应**：处理程序可以返回多种类型，这些类型可以转换为响应。

## 运行项目

要运行此项目，请导航到 `examples/stage5-projects/01-axum-web-app` 目录并运行以下命令：

```bash
cargo run
```

服务器将在 `http://127.0.0.1:3001` 上启动。
